## Projects

**Whatcoffee Point-of-Sale Database** — Designed and implemented a relational database for a local coffee shop in OMIS 105 at SCU. Built the ER model, normalized to 3NF, wrote SQL table creation scripts, and developed multi-table queries for order tracking, inventory, and revenue reporting. *(MySQL, SQL Workbench)*

**Order Fulfillment Accuracy Tracker** — Excel workbook analyzing personal fulfillment performance from my time as a Whole Foods Personal Shopper. Tracks weekly accuracy and substitution rates across 10 weeks, breaks down sub frequency by product category, and translates patterns into process improvement recommendations. *(Excel, root cause analysis)*
